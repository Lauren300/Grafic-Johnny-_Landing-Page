🖨️ Gráfica Johnny — Landing Page de Personalização

✨ Sobre o projeto

A Gráfica Johnny é uma landing page desenvolvida para apresentar serviços de personalização gráfica, com foco em empresas e clientes que buscam fortalecer sua identidade visual.

O projeto foi pensado para oferecer uma navegação clara, moderna e estratégica, destacando produtos e incentivando a conversão através de uma experiência intuitiva.

🧠 Problema

Muitas gráficas não possuem uma presença digital eficiente, dificultando:

A visualização dos produtos
A compreensão dos serviços oferecidos
A conversão de visitantes em clientes
🎯 Objetivo

Criar uma landing page que:

Destaque produtos personalizados de forma visual
Facilite a navegação e descoberta de serviços
Comunique profissionalismo e credibilidade
Incentive o usuário a solicitar ou adquirir produtos
🎨 Design (UX/UI)

O design foi estruturado com foco em:

🎯 Hierarquia visual clara (produtos em destaque)
🎨 Identidade visual consistente (cores fortes e modernas)
🧩 Organização por categorias (camisas, canecas, adesivos, etc.)
📱 Estrutura preparada para responsividade
🧩 Estrutura da página

A landing page foi organizada em seções estratégicas:

🔝 Header com navegação simples
🎯 Banner principal (Hero) com identidade da marca
🛍️ Categorias de produtos (ícones interativos)
🔥 Produtos mais populares
🏢 Seção institucional ("Personalize sua empresa")
❓ Seção de recomendação (o que você procura)
📞 Rodapé com informações e redes sociais
